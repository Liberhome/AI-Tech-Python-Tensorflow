```python
phones=['XiaoMi','HuaWei','Apple']
print(phones[0].title())
```

****
Add at the rear end
```python
phones.insert(0,'OPPO')
print(phones[0].title())
```


****
Add at anyplace
```python
phones.insert(0,'OPPO')
print(phones[0].title())
```

****
Delete at anyplace
```python
del phones[0]
print(phones[0].title())
```


```python
****
Pop out & use it
first_got=phones.pop(0)
print(first_got.title())
```
